For use with URP, please follow the instruction at http://leloctai.com/asset/translucentimage/docs/articles/universalrp.html

TLDR:
 - Import UniversalRP support.unitypackage
 - Add TranslucentImageBlurSource Renderer Features to each Forward Renderer Assets
 - For URP 12+, if the effect does not work correctly, switch Source Buffer on the Translucent Image Source component(s).
